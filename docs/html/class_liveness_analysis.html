Detailed Description

Compute liveness analysis information for a function.

This class performs backward data-flow analysis to determine the set of Value objects that are live at the beginning and at the end of a BasicBlock, respectively.

A PHINode defined and then used in the same block will not appear in the liveIn set for the block, as it is killed before it is used. A live Value can be either an Argument or an Instruction, but never a Constant.
