Detailed Description

Compensation code object.

This object can be used to encode local (i.e., required to reconstruct a specific Value) or global (i.e., to be executed at the beginning of the continuation function) compensation code.

A CompCode is essentially a lambda function that takes as arguments a subset of the live values transferred at an OSR point.

Member Data Documentation

CompCodeArgs* StateMap::CompCode::args

Arguments for the compensation code sequence.

Arguments are pointers to Value objects from the source function transferred during an OSR transition.

CodeSequence* StateMap::CompCode::code

Sequence of instructions to reconstruct the value.

In each instruction in the sequence, each operand can be a Value from args, a Constant, or a previous Instruction from the sequence.

llvm::Value* StateMap::CompCode::value

Value computed by the compensation code.

For local compensation code, it should correspond to the last instruction in code. For global compensation code, nullptr should be used.
